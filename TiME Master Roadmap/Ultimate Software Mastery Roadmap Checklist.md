# Mastery Roadmap Checklist (Zero → 20-Year Dev Level)

> **Organized by layers**, with increasing depth and real-world skill focus.
Designed to match **20 years of experience** in **~5 years**, 
with 12~ hours/day grind and **Project TAI** as a proving ground.

> ## 1. **Computer Science Fundamentals**

### Core CS Concepts

- [ ] Data Structures: Arrays, Linked Lists, Stacks, Queues, HashMaps, Trees, Graphs
- [ ] Algorithms: Sorting, Searching, Recursion, Dynamic Programming, Greedy
- [ ] Big O Notation & Time-Space Complexity
- [ ] Design Patterns (GOF, enterprise-level)
- [ ] OOP, SOLID Principles
- [ ] Functional Programming (Java, Kotlin, Swift, JS)

### Java (Core & Advanced)

- [ ] Java SE 17+: Streams, Generics, Collections
- [ ] Exception handling, I/O, Threads
- [ ] Lambdas & Functional Interfaces
- [ ] JVM internals & memory management
- [ ] Java 8+ features → Java 21 (recent changes)

### Kotlin

- [ ] Coroutines, Flows, Android-specific idioms
- [ ] Interop with Java

### Swift (for iOS)

- [ ] SwiftUI, Combine, Async/Await
- [ ] Protocol-oriented programming

### TypeScript & JavaScript

- [ ] Async/Await, Closures, Hoisting, Promises
- [ ] ES6+ syntax, classes, fetch APIs

### Core Frontend

- [ ] HTML5 + CSS3 + Responsive Design

- [ ] Accessibility (a11y) & SEO

- [ ] React or Vue (choose primary)
